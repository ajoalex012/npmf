
# NPMF

A better and open source search for node packages


## Run Locally

Clone the project

```bash
  git clone https://github.com/wearereadytocode/npmf
```

Go to the project directory

```bash
  cd npmf
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
